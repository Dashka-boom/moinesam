Проект для тренировки по примерному заданию ДЭ БУ.

1. В Open Server создать домен/папку проекта.
2. В phpMyAdmin импортировать moinesam.sql или создать таблицы вручную.
3. Название базы данных: moinesam.
4. Страница регистрации: registr.php
5. Страница входа: login.php
6. Администратор: логин adminka, пароль password.

Файлы:
- db.php — подключение к БД
- registr.php — регистрация
- login.php — авторизация
- request.php — история заявок пользователя
- create-request.php — создание заявки
- admin.php — панель администратора
- logout.php — выход
- style.css — стили
